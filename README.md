# Рецепты — приложение для управления кулинарными рецептами

Профессиональное веб-приложение для хранения и управления рецептами с бэкендом на FastAPI и SQLite.

## 🚀 Возможности

- ✨ Просмотр рецептов в виде элегантных карточек
- ➕ Добавление новых рецептов с автоматической генерацией SVG-иллюстраций
- ✏️ Редактирование существующих рецептов
- 🗑️ Удаление рецептов с подтверждением
- 🔍 Поиск по названию
- 🏷️ Фильтрация по категориям (Завтрак, Обед, Ужин, Десерт)
- 🎨 Две темы оформления (светлая/тёмная)
- 🤖 ИИ-генерация ингредиентов и инструкции по названию блюда
- 💾 Хранение данных в SQLite
- 📱 Полностью адаптивный дизайн

## 📋 Требования

- Python 3.8+
- pip

## 🔧 Установка и запуск

1. **Клонируйте или скачайте проект**

2. **Перейдите в папку проекта**

```bash
cd recipe_app
```

3. **Установите зависимости**

```bash
pip install -r requirements.txt
```

Или установите вручную:

```bash
pip install fastapi uvicorn python-multipart
```

4. **Запустите приложение**

```bash
uvicorn main:app --reload
```

5. **Откройте в браузере**

```text
http://localhost:8000
```

## 🛠️ API Эндпоинты

| Метод | Эндпоинт | Описание |
|--------|----------|----------|
| GET | `/api/recipes` | Получить все рецепты |
| POST | `/api/recipes` | Добавить новый рецепт |
| PUT | `/api/recipes/{id}` | Обновить рецепт |
| DELETE | `/api/recipes/{id}` | Удалить рецепт |

## 📝 Пример запроса (POST)

```json
{
  "name": "Паста карбонара",
```

## 🎨 Дизайн

- Современный минималистичный стиль с элементами стекломорфизма
- Шрифт **Inter** для отличной читаемости
- Адаптивная вёрстка для всех устройств
- Две цветовые схемы: светлая и тёмная

## 🤖 ИИ-генерация

При создании нового рецепта введите название блюда и нажмите кнопку **«✨ Сгенерировать»**. Приложение автоматически подберёт ингредиенты и пошаговую инструкцию на основе встроенной базы знаний.

## ⚡ Технологии

- **Бэкенд:** FastAPI, SQLite
- **Фронтенд:** Vanilla JS, HTML5, CSS3
- **Стилизация:** CSS-переменные, стекломорфизм
- **API:** RESTful с JSON-форматом
