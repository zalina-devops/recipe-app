# 🍽️ Рецепты — изысканная коллекция

Веб-приложение для хранения и управления кулинарными рецептами с бэкендом на FastAPI и SQLite. Стильный минималистичный дизайн, адаптивный интерфейс и удобное управление коллекцией рецептов.

## ✨ Возможности

- 📋 **Просмотр рецептов** — элегантные карточки с названием, кратким описанием и SVG-иллюстрацией
- ➕ **Добавление** — форма с полями: название, ингредиенты, инструкция, категория, время приготовления
- ✏️ **Редактирование** — возможность изменять любой существующий рецепт
- 🗑️ **Удаление** — кнопка «Удалить» с подтверждением для защиты от случайного стирания
- 🔍 **Поиск** — быстрый поиск по названию рецепта
- 🏷️ **Фильтрация** — фильтр по категориям (Завтрак, Обед, Ужин, Десерт)
- 🎨 **Две темы** — светлая и тёмная, переключение одной кнопкой
- 🤖 **Генерация рецептов** — введите название блюда и получите готовый рецепт из локальной базы (200+ рецептов)
- 👁️ **Просмотр деталей** — клик по карточке открывает полную информацию о рецепте
- 💾 **Хранение** — данные сохраняются в SQLite базе данных
- 📱 **Адаптивность** — корректно отображается на всех устройствах

# 📖 Пользовательское руководство

## Главная страница

После запуска приложения вы увидите:

- Заголовок **«Рецепты»** с подзаголовком **«изысканная коллекция»**
- Кнопку смены темы в правом верхнем углу
- Поле поиска по названию рецепта
- Фильтр по категориям
- Кнопку **«+ Добавить рецепт»**
- Сетку карточек с вашими рецептами

---

## Добавление рецепта

1. Нажмите кнопку **«+ Добавить рецепт»**.
2. Заполните поля:
   - **Название блюда** — например, *«Паста карбонара»*
   - **Ингредиенты** — список через запятую
   - **Инструкция** — пошаговое описание приготовления (каждый шаг с новой строки)
   - **Категория** — выберите из выпадающего списка
   - **Время приготовления** — укажите в минутах
3. Нажмите **«Сохранить»**.

> 💡 **Совет:** Используйте кнопку **«✨ Сгенерировать»**, чтобы автоматически заполнить поля рецептом из базы.

---

## Просмотр рецепта

1. Кликните по любой карточке рецепта.
2. Откроется модальное окно с полной информацией:
   - Название блюда
   - Категория и время приготовления
   - Полный список ингредиентов
   - Пошаговая инструкция
3. Нажмите **«Закрыть»**, чтобы вернуться к списку.

---

## Редактирование рецепта

### Способ 1: Через карточку

1. На карточке рецепта нажмите **«Редактировать»**.
2. Внесите изменения в открывшейся форме.
3. Нажмите **«Сохранить»**.

### Способ 2: Через просмотр

1. Откройте рецепт (клик по карточке).
2. В окне просмотра нажмите **«Редактировать»**.
3. Внесите изменения и сохраните.

---

## Удаление рецепта

1. На карточке рецепта нажмите **«Удалить»**.
2. Подтвердите удаление в диалоговом окне.
3. Рецепт будет безвозвратно удалён.

---

## Поиск и фильтрация

### Поиск

Начните вводить название блюда в поле **«Поиск по названию»** — список обновится автоматически.

### Фильтрация

Выберите категорию в выпадающем списке — останутся только рецепты выбранной категории.

---

## Темы оформления

Нажмите кнопку **«Сменить тему»** в правом верхнем углу для переключения между светлой и тёмной темой.

Выбранная тема сохраняется в браузере.

---

## Генерация рецептов

1. Откройте форму добавления рецепта.
2. Введите название блюда в поле **«Название блюда»**.
3. Нажмите кнопку **«✨ Сгенерировать»**.
4. Поля **«Ингредиенты»** и **«Инструкция»** заполнятся автоматически.
5. При необходимости отредактируйте результат вручную.
6. Нажмите **«Сохранить»**.

> 📌 **Примечание:** Генерация использует локальную базу из 200+ рецептов. Если блюдо не найдено, вы получите предложение похожего рецепта.

# 🛠️ API Эндпоинты

| Метод | Эндпоинт | Описание |
|-------|----------|----------|
| GET | `/api/recipes` | Получить все рецепты |
| POST | `/api/recipes` | Добавить новый рецепт |
| PUT | `/api/recipes/{id}` | Обновить существующий рецепт |
| DELETE | `/api/recipes/{id}` | Удалить рецепт |

### Пример запроса (POST)

```json
{
  "name": "Паста карбонара",
  "ingredients": "Спагетти, яйца, панчетта, пармезан, чеснок, оливковое масло",
  "instructions": "1. Отварить спагетти до al dente.\n2. Обжарить панчетту с чесноком.\n3. Смешать с яйцами и пармезаном.",
  "category": "Ужин",
  "time": 20
}
```
